해쉬맵을 이용해 참가 선수들에게 각각 1의 value값을 부여하고 이름이 겹치는 경우에 +1을 해주었습니다. 완주했을 경우에는 그 선수의 value에-1을 해주어 결과적으로 value값이 0이 아닌 선수가 완료하지 못한 선수입니다.
